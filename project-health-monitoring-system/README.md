Project Health Monitoring & Risk Alert System
Overview

This project is a multi-project tracking and automated risk detection system built using Google Sheets and Apps Script.

It enables real-time monitoring of project timelines, budget variance, and completion status while automatically flagging delayed projects through controlled email alerts.

Tech Stack:

Google Sheets (Project Tracker & Dashboard)

Google Apps Script (Automation Engine)

Conditional Formatting (Visual Risk Indicators)

Key Features:

Deadline-based delay detection

Budget variance monitoring

Dynamic project status classification

Automated email alerts with duplicate-notification control

Risk log sheet for historical issue tracking

Executive-style dashboard for multi-project visibility

System Workflow:

Project data entered into structured tracker

Status logic evaluates delay and budget conditions

If status = “Delayed” and not previously notified:

Email alert is triggered

Notification column updated

Risk logged in separate sheet

Dashboard Metrics:

Total Projects

On Track vs Delayed Distribution

Budget Overruns

Average Completion Rate
